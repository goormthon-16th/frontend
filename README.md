## 4. 핵심 기술 및 아키텍처

### Front-end
| 기술명 | 선정 이유 |
| :--- | :--- |
| **Next.js** | Next.js의 서버 사이드 렌더링(SSR) 및 정적 생성 기능을 활용하여, 검색 엔진 최적화와 빠른 애플리케이션 성능을 동시에 확보했습니다. |
| **JavaScript** | 신속한 MVP 개발 및 배포를 위해 TypeScript 대신 설정 및 컴파일 과정이 적은 JavaScript를 채택하였습니다. |
| **Vapor** | 구름 디자인 시스템을 기반으로 하여 UI의 디자인 통일성을 확보하고 개발 생산성을 높였습니다. |

### Back-end
| 기술명 | 선정이유 |
| :--- | :--- |
| **SpringBoot** | Rest API 구축을 위한 백엔드 설계를 진행했습니다. |
| **MySQL** | 데이터의 안정적인 저장과 관리를 위한 관계형 데이터베이스를 구축했습니다. |
| **Kubernetes(EKS)** | EKS 환경에서 프론트엔드, 백엔드, DB를 배포했습니다. |
| **Jenkins** | 애플리케이션 코드를 빌드하고, 컨테이너 이미지를 생성하는 CI를 진행했습니다. |
| **ArgoCD** | Git 저장소의 변경사항을 쿠버네티스 클러스터에 동기화했습니다. |
